# Health Claims Actuarial Analysis Project

## 📖 Overview
This is an ongoing project where I am exploring how **SQL** and **Tableau** can be combined to analyze health claims data from an actuarial perspective.  
The goal is to connect technical data analysis with actuarial concepts such as experience studies, risk classification, loss ratio analysis, medical cost trends, and model governance.

---

## ⚙️ Current Status
🚧 **Work in Progress**  
I am currently building SQL queries and Tableau dashboards to demonstrate actuarial-style insights.  
Future updates will include more refined dashboards, Python scripts for validation, and extended documentation.

---

## 📊 Analysis Components
1. **Claims Frequency & Severity**  
   - SQL: Count and aggregate claims by procedure and diagnosis  
   - Tableau: Trend and distribution charts  
   - Actuarial link: Experience study for pricing and reserving assumptions  

2. **Risk Classification**  
   - SQL: Segment claims by age, gender, diagnosis, and place of service  
   - Tableau: Comparative dashboards  
   - Actuarial link: Rating factors and risk adjustment simulation  

3. **Loss Ratio Analysis**  
   - SQL: Calculate claims-to-premium (or charges) ratios  
   - Tableau: Profitability trends by category  
   - Actuarial link: Pricing and reinsurance decisions  

4. **Medical Cost Trend Detection**  
   - SQL: Aggregate claims by month/quarter  
   - Tableau: Time-series visualization  
   - Actuarial link: Forecasting and reserving  

5. **Model Governance Checks**  
   - SQL: Reconciliation between charges, payments, adjustments, and outstanding balances  
   - Tableau: Data quality dashboards  
   - Actuarial link: Model validation and governance practices
